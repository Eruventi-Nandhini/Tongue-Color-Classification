Tongue Color Classification using Deep Learning (TCM)
📌 Overview
This project uses deep learning to classify tongue colors based on Traditional Chinese Medicine (TCM) concepts. It helps in identifying possible health conditions from tongue images.
🎯 Objective
To build an AI model that predicts tongue color automatically from images.
🧠 Technologies
Python
TensorFlow / Keras
OpenCV
HTML, CSS
Flask
⚙️ How to Run
Install dependencies:
pip install -r requirements.txt
Run project:
python app.py
Open browser:
http://127.0.0.1:5000/
🖼️ Features
Upload image
Predict tongue color
Simple UI
📊 Output Classes
Red
Light Red
Dark Red
Purple
⚠️ Note
Dataset and trained model are not included due to large size.
